# Tool Rules
The app should use the following tools consistently and use the most recent versions:
- **pnpm** (very important)
- **nextJS**
- **tailwind**
- **prisma**
- **biome** (for linting and formatting)
- **better-auth**
- **shadcn / ui** (use most recent cli to create components)
- **Tanstack Query**
- **Tanstack Table**
- **react hook form**
- **next-safe-action** (every server action is secured by next-safe-action handler)
- **nuqs**: for states of tabs, tables and other components
- **docker** for local development and deployment
- **next-intl** for translations (all texts are created with translation keys)
- **tsc** use tsc to typecheck after implementing stuff (not next build)
